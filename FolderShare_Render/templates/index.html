📁 FolderShare

Instructies

  1. Klik op Selecteer map.
  2. Kies de map die je wilt delen.
  3. De browser selecteert automatisch alle bestanden in die map.
  4. Klik op Uploaden.
  5. Na het uploaden krijg je een share-code.
  6. Stuur die code naar iemand anders.

⚠️ Werkt map-selectie niet? Maak dan eerst een ZIP-bestand van de map en upload dat ZIP-bestand.
